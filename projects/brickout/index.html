BrickOut

PROJECT TIMELINE - December 2019 to Present
ROLES - Software Developer, Game Developer, Web Developer
SKILLS INCORPORATED - Software Development, Web Design and Development, Game Design and Development
SOFTWARE USED - Phaser 3, TypeScript, HTML / CSS, Adobe Illustrator, Adobe Photoshop

A classic brick breaker game with different types of power-ups in the game. The main purpose of this project is to showcase proper software design and development principles and the application of Phaser 3 with TypeScript. This project is in its alpha stage, so please expect some bugs if trying to play the game.

You can view the project code on my GitHub page.
