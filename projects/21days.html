21 DAYS

PROJECT TIMELINE - February 2020 to Present
ROLES - Game Designer, Level Designer, Gameplay Programmer, UI / UX Designer
SKILLS INCORPORATED - Game Mechanism and Systems Design, Level Design, Gameplay Programming, Software Development, UI / UX Design
SOFTWARE USED - Unity, Adobe Illustrator, Adobe Photoshop, Milanote
LINKS - View this project on GitHub. View my development board on Milanote.

This is the latest game design project I have started working on, while I look for a full time job. I intend to release it on iOS / Steam in the near future when this game finishes development.

21 Days is an escape the maze type of puzzle game which includes a horror theme to it. You are trapped in a never ending house (maze), and you have 21 Days to find your way out of it. There are extraterrestrial beings wandering around in the house and you have to sneak around them in order to escape the house. There are also some items that may help you deal with these beings by stunning them or damaging them etc.

What makes this game unique is that, in this game, time is money. Literally! Every action in this game, including walking, costs time and the more actions you take, the lesser amount of time you have to escape the maze. In other words, you really have 21 days to escape the house!

This game is currently in its early development phase, and includes a level blockmesh for the first / tutorial level of the game, which is also what I am currently working on. You can expect to see the complete level and a UI design in this game very soon!

You can check learn more about this game, including its Game Design, Level Design and Software Development aspects and read through what I am doing for the different stages of development currently on my project board for Milanote.
